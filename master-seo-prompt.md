## 📐 Algorithmic Execution Rules

### 1. Formatting Matrix
*   **Hierarchy:** Strict Markdown sequence. Only one primary `# H1` per document. Core topics utilize `## H2`, micro-features utilize `### H3`.
*   **Scannability:** Paragraph blocks must never exceed 3 sentences. Bold key takeaways to guide speed readers.

### 2. Semantic Token Controls (Anti-AI Filter)
*   **Style Pattern:** Avoid repetitive AI clichés and unnatural transition words such as *in conclusion, delve, testament, moreover, furthermore, tapestry, look no further, beacon,* or *nestled*.
*   **Permitted Cadence:** Use natural human transitions like *Truth is, Here is why, Think about it,* or *That means*.

### 3. E-E-A-T Core Integration
Ensure the content structurally demonstrates Google’s core quality principles:
*   **Experience:** Reference first-hand observation of brewing metrics, bean textures, and raw roasting workflows.
*   **Expertise:** Use professional coffee industry terminology correctly (e.g., extraction time, TDS levels, Arabica varietals, crumb structure).
*   **Authoritativeness:** Position the café as a local domain leader that values meticulous culinary sourcing over mass bulk processing.
*   **Trustworthiness:** Provide transparent details regarding baking fresh on-site and coffee processing timelines.

### 4. Localization Intercepts
*   Dynamically interweave local geographic tokens (`Boring Road Crossing`, `Patliputra Colony`, `Kankarbagh`, `Patna University students`, `Bihar's tech creators`) natively within contextual narratives. Avoid raw keyword stuffing.
